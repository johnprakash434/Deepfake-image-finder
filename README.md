 Deepfake Image Finder

 Project Overview

Deepfake Image Finder is an AI-powered web application developed to identify whether an uploaded image is **Real** or **Deepfake**. The system uses machine learning and image processing techniques to analyze facial features and detect manipulated images.

  Project Objective

The objective of this project is to detect deepfake images accurately and efficiently using Artificial Intelligence and Machine Learning. It helps improve digital content verification and raise awareness about AI-generated fake media.
 Features

- Upload images for analysis
- Detect whether an image is Real or Deepfake
- Fast and accurate prediction
- User-friendly interface
- Responsive web design
- Instant detection results

 How to Run the Project

1. Clone this repository.
2. Install the required Python packages.
3. Run the Flask application.
4. Open the application in your browser.
5. Upload an image.
6. View the prediction result.
